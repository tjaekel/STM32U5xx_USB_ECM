The The STM32U5 series offers advanced power-saving microcontrollers, based on Arm® Cortex®-M33 to meet the most demanding power/performance requirements for smart applications, including wearables, personal medical devices, home automation, and industrial sensors. Offering up to 2 Mbytes of Flash (dual bank) memory and 786 Kbytes of SRAM, the STM32U5 series of microcontrollers takes performance to the next level. The STM32U5 offers a large portfolio with 8 packages (from 48 to 169 pins) and supports up to 125°C ambient temperature.


The The STM32U5xx devices are an ultra-low-power microcontrollers family (STM32U5 Series) based on the high-performance Arm® Cortex®-M33 32-bit RISC core. They operate at a frequency of up to 160 MHz.

 

 About this application

 

This application is part of the Azure RTOS STM32Cube expansion package for STM32U5 series, it presents HTTP webserver developed based on the USBX CDC ECM class, Azure RTOS NetX Duo TCP/IP network stack and FileX file system stack, it allows the loading of a static home web page from sd card and led control web page.

Application main features:

  • Azure RTOS USBX
      • Class: Device CDC ECM.
  • Azure RTOS NetX Duo
      • Protocols: IP, ICMP, ARP, UDP, TCP.
      DHCP client (dynamic IP address allocation).
      Addons/Web HTTP.
  • Azure RTOS FileX API (to fetch the html page from sd card).


For more information you can refer to Azure RTOS documentation  Azure RTOS USBX & Azure RTOS NetX Duo & Azure RTOS FileX
